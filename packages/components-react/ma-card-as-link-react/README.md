# Card As Link

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-card-as-link-react
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-card-as-link-react
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-card-as-link-react
```

## Usage

If you just want the React component

```typescript
import { CardAsLink } from '@nl-design-system-community/ma-card-as-link-react/dist/ma-card-as-link.mjs';
```
