# Task List

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-task-list-react
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-task-list-react
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-task-list-react
```

## Usage

```typescript
import { TaskList } from '@nl-design-system-community/ma-task-list-react/dist/ma-task-list.mjs';
```
