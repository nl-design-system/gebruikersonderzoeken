# Tag

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-tag-react
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-tag-react
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-tag-react
```

## Usage

```typescript
import { Tag } from '@nl-design-system-community/ma-tag-react/dist/ma-tag.mjs';
```
