# Page Footer

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-page-footer-react
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-page-footer-react
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-page-footer-react
```

## Usage

If you just want the React component

```typescript
import { PageFooter } from '@nl-design-system-community/ma-page-footer-react/dist/ma-page-footer.mjs';
```
