# Card List

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-card-list-react
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-card-list-react
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-card-list-react
```

## Usage

If you just want the React component

```typescript
import { CardList } from '@nl-design-system-community/ma-card-list-react/dist/ma-card-list.js';
```
