# Page Header

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-page-header-react
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-page-header-react
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-page-header-react
```

## Usage

```typescript
import { PageHeader } from '@nl-design-system-community/ma-page-header-react/dist/ma-page-header.mjs';
```
