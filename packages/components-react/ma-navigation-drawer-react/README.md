# Navigation Drawer

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-navigation-drawer-react
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-navigation-drawer-react
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-navigation-drawer-react
```

## Usage

```typescript
import { NavigationDrawer } from '@nl-design-system-community/ma-navigation-drawer-react/dist/ma-navigation-drawer.mjs';
```
