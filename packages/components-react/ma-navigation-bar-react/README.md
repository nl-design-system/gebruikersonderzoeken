# Navigation Bar

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-navigation-bar-react
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-navigation-bar-react
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-navigation-bar-react
```

## Usage

If you just want the React component

```typescript
import { NavigationBar } from '@nl-design-system-community/ma-navigation-bar-react';
```
