# MA Focus Indicator

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-focus-indicator-css
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-focus-indicator-css
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-focus-indicator-css
```
