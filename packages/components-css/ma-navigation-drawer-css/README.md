# MA Navigation Drawer

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-navigation-drawer-css
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-navigation-drawer-css
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-navigation-drawer-css
```
