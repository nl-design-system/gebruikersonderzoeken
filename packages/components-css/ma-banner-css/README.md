# MA Banner

De banner van een website

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-banner-css
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-banner-css
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-banner-css
```
