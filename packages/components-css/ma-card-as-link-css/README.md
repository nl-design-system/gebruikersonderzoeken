# MA Card As Link

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-card-as-link-css
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-card-as-link-css
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-card-as-link-css
```
