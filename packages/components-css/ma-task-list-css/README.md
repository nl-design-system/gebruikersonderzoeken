# MA Task List



## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-task-list-css
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-task-list-css
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-task-list-css
```
