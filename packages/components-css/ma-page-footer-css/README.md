# MA Page Footer

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-page-footer-css
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-page-footer-css
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-page-footer-css
```
