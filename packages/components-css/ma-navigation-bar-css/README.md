# MA Navigation Bar

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-navigation-bar-css
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-navigation-bar-css
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-navigation-bar-css
```
