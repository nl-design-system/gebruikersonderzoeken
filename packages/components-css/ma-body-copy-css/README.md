# MA Body Copy

Style de copy volgens NL Design System styling

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-body-copy-css
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-body-copy-css
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-body-copy-css
```
