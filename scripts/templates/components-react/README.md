# New Component

## Installation

Using npm

```shell
npm install @nl-design-system-community/ma-new-component-react
```

Using pnpm

```shell
pnpm add @nl-design-system-community/ma-new-component-react
```

Using yarn

```shell
yarn add @nl-design-system-community/ma-new-component-react
```

## Usage

If you just want the React component

```typescript
import { NewComponent } from '@nl-design-system-community/ma-new-component-react';
```

If you want the React component with styles

```typescript
import { NewComponent } from '@nl-design-system-community/ma-new-component-react/css';
```
