# Colofon

## Over deze website

De website wordt gemaakt en onderhouden door de NL Design System community en is nog volop in ontwikkeling. Mocht je vragen of opmerkingen hebben dan kun je contact met het kernteam van NL Design System opnemen op het centrale e-mailadres: <a href="mailto:info@nldesignsystem.nl">info@nldesignsystem.nl</a>.

## Open source

Bij [NL Design System](https://nldesignsystem.nl) wordt alles zoveel mogelijk open source beschikbaar gemaakt. Zo is samenwerken makkelijker, is de code herbruikbaar over organisatiegrenzen heen en kunnen inzichten en documentatie eenvoudig worden gebruikt, uitgebreid en verbeterd. De manier om dat te bereiken is met de volgende Open Source licenties.

### EUPL 1.2

Alle componenten in code en Figma die gebruikt zijn om deze website op te bouwen zijn beschikbaar onder de [EUPL 1.2](https://nldesignsystem.nl/open-source/eupl) licentie.

### CC0 1.0 Universeel

De gedeelde onderzoeken op gebruikersonderzoeken.nl en de beschikbare documentatie is gepubliceerd onder de [CC0 1.0 Universeel](https://nldesignsystem.nl/open-source/cc0) licentie.

**Let op** Deze licentie is uitsluitend van toepassing op de gedeelde teksten die op de website worden getoond met uitzondering van het merkenrecht zoals een logo. De overige auteursrechten (van derden) zoals van het volledige rapport waar de hyperlink naar verwijst, en auteursrecht op afbeeldingen, foto's en video's blijven onverlet gelden.

Meer lezen over de keuze voor deze licenties? Renate schreef in 2024 een leuke blog: [Over de licentiekeuze van NL Design System](https://nldesignsystem.nl/blog/licentiekeuze-nl-design-system).

#### Hoe zit het dan met auteursrecht op het oorspronkelijke rapport?

De inzichten die gedeeld worden op gebruikersonderzoeken.nl zijn gebaseerd op onderzoeksrapporten, maar zijn vaak niet 1-op-1 overgenomen. Het oorspronkelijke onderzoeksrapport staat waar mogelijk in een link op de website, maar is uitgezonderd van de CC0 licentie die geldt op de inzichten die op de website zijn gedeeld. Zo kunnen anderen zich wel laten inspireren door het hele rapport en ervan leren, maar deze niet hergebruiken of delen zonder toestemming.
