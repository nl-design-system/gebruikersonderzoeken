# Colofon

## Over deze website

De website wordt gemaakt en onderhouden door de NL Design System community en is nog volop in ontwikkeling. Mocht u vragen of opmerkingen hebben dan kunt u contact met het kernteam van NL Design System opnemen op het centrale e-mailadres: <a href="mailto:info@nldesignsystem.nl">info@nldesignsystem.nl</a>.

## Open source

Bij [NL Design System](https://nldesigsysytem.nl) wordt alles zoveel mogelijk open source beschikbaar gemaakt. Zo kan makkelijker worden samengewerkt, is de code herbruikbaar over organisatiegrenzen heen en kunnen inzichten en documentatie eenvoudig worden gebruikt, uitgebreid en verbeterd. De manier om dat te bereiken: met Open Source licenties!

- [EUPL 1.2](../open-source/eupl.md) voor code en Figma componenten.
- [CC0 1.0 Universeel](../open-source/cc0.md) voor documentatie en inzichten uit gebruikersonderzoeken.

Meer lezen over de keuze voor deze licenties? Renate schreef in 2024 een leuke blog: [Over de licentiekeuze van NL Design System](blog/licentiekeuze-nl-design-system).

### Hoe zit het dan met auteursrecht op het oorspronkelijke rapport?

De inzichten die gedeeld worden op gebruikersonderzoeken.nl zijn gebaseerd op onderzoeksrapporten, maar zijn vaak niet 1-op-1 overgenomen. Het oorspronkelijke onderzoeksrapport staat waar mogelijk in een link op de website, maar is uitgezonderd van de CC0 licentie die geldt op de inzichten die op de website zijn gedeeld. Zo kunnen anderen zich wel laten inspireren door het hele rapport en ervan leren, maar deze niet hergebruiken of delen zonder toestemming.
