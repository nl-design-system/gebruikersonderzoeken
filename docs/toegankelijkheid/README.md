# Toegankelijkheid Gebruikersonderzoeken.nl

We streven ernaar voor de website gebruikersonderzoeken.nl te voldoen aan de [WCAG 2.2 AA richtlijnen](https://www.w3.org/TR/WCAG22/), zoals opgesteld door het W3C. We publiceren periodiek een rapport dat voldoet aan [de eisen van DigiToegankelijk](https://www.digitoegankelijk.nl/toegankelijkheidsverklaring/controle).

Ondervind je een probleem bij het gebruik van deze website of heb je hier een vraag over?
Neem dan [contact](mailto:info@nldesignsystem.nl) met ons op.
